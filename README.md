# Todo List App

A simple todo list application that allows users to add, remove, and manage tasks. Built to practice core frontend concepts like state management, DOM manipulation, and event handling.

---

## Installation

Clone the repository:

```sh
git clone <your-repo-link>
cd <your-project-folder>
```

Install dependencies:

```sh
npm install
```
---

## Running the Development Server

Start the app:

```sh
npm run dev
```

Open in browser:

```
http://localhost:5173
```
