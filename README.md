# Todo Dashboard

A modern React-based task management application that allows authenticated users to create, edit, organize, search, filter, and track personal tasks through a responsive and intuitive user interface.

## Features

* User authentication (login and logout)
* Protected routes using React Router
* Create new todos
* Edit existing todos
* Mark todos as completed
* View active, completed, or all todos
* Search todos by title
* Sort todos by creation date or title
* Sort ascending or descending
* Responsive design for desktop, tablet, and mobile devices
* Loading states during API requests
* Error handling and recovery
* Empty state messaging
* DOMPurify input sanitization
* Accessible forms and navigation controls

## Getting Started

### Prerequisites

Before running this project, ensure you have:

* Node.js (v18 or newer recommended)
* npm

### Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd ctd-todo-list
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Design Decisions

The application uses a clean dashboard-inspired design built around a consistent color palette and reusable UI patterns.

### Design Goals

* Clear visual hierarchy
* High readability and accessibility
* Responsive experience across devices
* Consistent component styling
* Minimal distractions for task management

### Styling Approach

* CSS Modules for component-scoped styling
* Mobile-first responsive layouts
* Card-based content organization
* Accessible focus states and touch targets
* Consistent spacing and typography system
